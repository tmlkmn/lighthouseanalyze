💡 Öncelikli İyileştirme Önerileri

Performansı artırmak için öncelik sırasına göre öneriler.

YÜKSEK ÖNCELİK
1. Video Dosyalarını Optimize Edin
Sayfada toplam ~47 MB video dosyası yüklenmektedir. Videoları sıkıştırın, WebM/AVIF formatına dönüştürün ve lazy loading uygulayın. Viewport dışındaki videolar için poster image kullanın ve kullanıcı etkileşimine kadar yüklemeyi erteleyin.
YÜKSEK ÖNCELİK
2. Kullanılmayan CSS'i Kaldırın
~936 KB kullanılmayan CSS tespit edildi. CSS dosyalarını critical/non-critical olarak ayırın. İlk ekranda gerekli olmayan stilleri async yükleyin. PurgeCSS gibi araçlar kullanın.
YÜKSEK ÖNCELİK
3. Kullanılmayan JavaScript'i Azaltın
~1.133 KB kullanılmayan JavaScript tespit edildi. Code splitting uygulayın, kullanılmayan modülleri kaldırın ve dinamik import kullanın. Tree shaking'i etkinleştirin.
YÜKSEK ÖNCELİK
4. Render Engelleyen Kaynakları Ortadan Kaldırın
~940 ms (masaüstü) / ~5.880 ms (mobil) tasarruf mümkün. CSS'i kritik ve kritik olmayan olarak ayırın. JavaScript'lere defer/async ekleyin. Kritik CSS'i inline edin.
ORTA ÖNCELİK
5. CSS ve JavaScript Dosyalarını Küçültün (Minify)
CSS'de ~180 KB, JavaScript'te ~165 KB tasarruf mümkün. Build sürecinize minification adımı ekleyin. Terser ve cssnano gibi araçlar kullanın.
ORTA ÖNCELİK
6. HTTP/2 Protokolüne Geçin
Sunucu HTTP/1.1 kullanıyor. HTTP/2 multiplexing ile paralel kaynak yükleme sağlanabilir. ~160 ms (masaüstü) / ~490 ms (mobil) tasarruf mümkün.
ORTA ÖNCELİK
7. Önbellek Stratejisi İyileştirin
~27 KB (masaüstü) / ~34 KB (mobil) önbellek tasarrufu mümkün. Statik kaynaklar için uzun süreli cache-control başlıkları ayarlayın. CDN kullanımını optimize edin.
ORTA ÖNCELİK
8. Görsel Dosyaları Optimize Edin
~10 MB görsel dosya tespit edildi. WebP/AVIF formatına dönüştürün, responsive images kullanın, lazy loading ekleyin. faslider.png (~5.6 MB) özellikle kritik.
DÜŞÜK ÖNCELİK
9. Erişilebilirlik Sorunlarını Düzeltin
Bağlantılara ayırt edilebilir ad ekleyin, dokunma hedeflerinin boyutlarını artırın (en az 48x48px). Mobilde ana landmark bölge eksik — <main> etiketi ekleyin.
DÜŞÜK ÖNCELİK
10. Kaynak Eşlemelerini (Source Maps) Ekleyin
Birinci taraf JavaScript dosyalarında kaynak eşlemeleri eksik. Bu, hata ayıklamayı zorlaştırır. Build sürecinize source map oluşturmayı ekleyin.
